# 🍔 Bites & Vibes Restaurant Web App

## 📌 Project Description
This is a restaurant web application that allows users to browse menu items, add them to cart, and view order history.

## 🚀 Features
- Display menu items dynamically using JavaScript
- Add items to cart with real-time cart counter update
- Navigate to item details page
- Store and retrieve orders using localStorage
- Clear order history functionality
- Contact form with input validation
- Dynamic success and error messages (UI updates)
- Dark mode toggle with user preference (Cookies)
- Responsive and user-friendly interface
- Event-driven interactions (click, submit, page load)
